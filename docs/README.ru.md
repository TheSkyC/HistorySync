# HistorySync
**HistorySync** — это мощное кроссплатформенное настольное приложение. Оно предоставляет комплексное и эффективное решение для единого управления историей браузеров и облачного резервного копирования. От сбора данных из множества браузеров и полнотекстового поиска за миллисекунды до автоматического резервного копирования WebDAV и подробной статистики — оно дает вам полный контроль над вашим цифровым следом.

Приложение изначально поддерживает базы данных браузеров на базе Chromium, Firefox и Safari, предлагая исключительную защиту конфиденциальности и удобное локальное управление.

## 🚀 Основные возможности

### 📂 Универсальный сбор данных (Поддержка 30+ браузеров)
*   **Огромная совместимость**: Встроенная поддержка Chrome, Edge, Firefox, Safari, Brave, Vivaldi, Arc и множества региональных/пользовательских браузеров (QQ, Sogou, CentBrowser и др.).
*   **Умное инкрементное извлечение**: Безопасное чтение снимков SQLite WAL, позволяющее извлекать данные без потерь и конфликтов даже во время работы браузеров.
*   **Импорт портативных БД**: Ручной импорт отдельных файлов `History` или `places.sqlite` для легкого объединения данных со старых компьютеров.

### 🔍 Быстрый поиск в стиле Spotlight и база знаний
*   **Оверлей быстрого доступа**: Нажмите `Ctrl+Shift+H` в любом месте, чтобы вызвать минималистичный оверлей поиска.
*   **Новый движок горячих клавиш**: Кроссплатформенная система на базе `pynput` с 14 настраиваемыми глобальными и внутренними шорткатами.
*   **Продвинутый синтаксис запросов**: Ищите как профи, используя токены (например, `domain:github.com`, `after:2024-01-01`).
*   **Закладки и аннотации**: Превратите свою историю в базу знаний. Добавляйте теги и текстовые заметки к важным страницам.

### ⚡ Высочайшая производительность и современный UI
*   **Плавная прокрутка миллионов записей**: Переписанная логика пагинации с использованием Keyset-индексов. Поиск по регулярным выражениям перенесен на уровень SQL.
*   **Адаптивный интерфейс**: Пропорциональное распределение ширины столбцов. Бесшовная поддержка переключения темной/светлой темы системы в реальном времени.
*   **Богатая визуализация данных**: Изучайте свои привычки с помощью тепловой карты в стиле GitHub, круговых диаграмм и 24-часовых графиков активности.

### ☁️ Облачная синхронизация и автоматизация
*   **Резервное копирование и слияние WebDAV**: Использует **атомарную потоковую загрузку**. При восстановлении из облака система интеллектуально объединяет записи с разных устройств.
*   **Headless CLI (`hsync`)**: Полнофункциональный инструмент командной строки для продвинутых пользователей.
*   **Тихий фоновый режим**: Работает свернутым в системном трее, автоматически выполняя извлечение и резервное копирование по расписанию.

### 🛡️ Максимальная конфиденциальность и контроль
*   **Скрытый режим и мягкое скрытие**: Выделенный вид "Скрытые записи". Поддержка мягкого скрытия определенных доменов (записи остаются в БД, но исчезают из главного вида).
*   **Архитектура безопасности V2**: Защищает конфиденциальные настройки с помощью независимых подключей шифрования и аутентификации HKDF.
*   **Черный список доменов**: Блокировка доменов в один клик.

## 🛠️ Настройка среды разработки

### Требования
*   Python 3.10 или выше
*   Git (опционально, для клонирования репозитория)

### Шаги
1.  **Клонируйте репозиторий (или скачайте ZIP)**
    ```bash
    git clone https://github.com/TheSkyC/HistorySync.git
    cd HistorySync
    ```

2.  **Создайте и активируйте виртуальную среду (Рекомендуется)**
    ```bash
    python -m venv venv
    # Windows
    .\venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    ```

3.  **Установите зависимости**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запуск**
    ```bash
    python -m src.main
    ```

## 🚀 Быстрый старт

HistorySync предлагает гибкие режимы работы:

### 1. 🔄 Тихий фоновый режим (Рекомендуется)
*Идеально для пользователей, которые хотят "настроить и забыть".*
1.  **Запуск**: Перейдите в `Настройки > Запуск` и включите "Запускать при старте системы".
2.  **Расписание**: Установите интервал извлечения в разделе `Автосинхронизация`.
3.  **Облако**: Введите данные WebDAV и включите авто-резервное копирование.
4.  **Работа**: Закройте главное окно. Приложение свернется в трей и будет тихо защищать ваши данные.

### 2. 🔍 Режим активного управления
*Идеально для частого поиска, создания заметок или очистки данных.*
1.  **Быстрый поиск**: Нажмите `Ctrl+Shift+H` в любом месте.
2.  **База знаний**: Добавляйте страницы в закладки и пишите заметки.
3.  **Конфиденциальность**: Удаляйте ненужные записи или используйте "Черный список доменов".
